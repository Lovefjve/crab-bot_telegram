# Crab Bot Telegram

Bot Telegram viết bằng Node.js theo kiến trúc command-based.

## Tài liệu chính

- Xem tài liệu kỹ thuật đầy đủ tại: [`PROJECT_INFO.md`](./PROJECT_INFO.md)

## Chạy dự án

```bash
npm install
npm start
```

## Ghi chú

- Entry point: `bot.js`
- Các lệnh nằm trong thư mục `commands/`
- Dữ liệu local nằm trong `data/` và `src-api/`
